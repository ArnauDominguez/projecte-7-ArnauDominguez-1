## 1. Visió General del Projecte
S'ha dissenyat i desenvolupat una solució web corporativa d'alta gamma per a **FoodLogistic**, l'empresa referent en distribució alimentària nacional. L'objectiu principal ha estat transmetre **seguretat, higiene i eficiència operativa** mitjançant una interfície neta, professional i minimalista.

## 2. Arquitectura de l'Estructura (HTML5 Semàntic)
El lloc s'ha construït seguint una jerarquia lògica per maximitzar el **SEO** i l'accessibilitat:

* **Header:** Navegació fixa per a un accés constant a les seccions.
* **Inici (Hero):** Secció d'impacte visual amb la proposta de valor principal.
* **Serveis:** Graella interactiva que detalla les línies de negoci.
* **Sobre nosaltres:** Contingut corporatiu per reforçar l'autoritat de la marca.
* **Contacte:** Formulari d'alta conversió per a la captació de clients potencials.

## 3. Especificacions Tècniques de Disseny (CSS3)
L'estètica se centra en un enfocament **Minimalista i Elegant**:

| Element | Descripció Tècnica |
| :--- | :--- |
| **Paleta de Colors** | Blau fosc corporatiu (`#1a2a3a`) i Taronja d'acció (`#e67e22`). |
| **Tipografia** | San-serif moderna per a una llegibilitat òptima en pantalles. |
| **Layout** | Sistema Híbrid **CSS Grid** i **Flexbox** per a total adaptabilitat. |
| **Responsive** | Disseny *Mobile-First* (optimitzat per a smartphones, tablets i desktop). |

## 4. Funcionalitat i Lògica (Vanilla JavaScript)
S'ha prioritzat el rendiment eliminant dependències externes. El script gestiona:

1. **Navegació Suau:** Implementació de *smooth scroll* per a una experiència d'usuari fluida.
2. **Validació de Formulari:** Control de dades en els camps de *Nom, Email, Telèfon i Missatge*.
3. **Interactivitat del Header:** Canvi visual de la barra de navegació segons el desplaçament (scroll).
4. **Feedback d'Usuari:** Sistema de confirmació d'enviament de dades simulat.

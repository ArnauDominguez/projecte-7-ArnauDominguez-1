# T10: Pressupost del projecte

## Introducció
En aquesta fase es presenta la proposta econòmica del projecte de modernització de FoodLogístic S.A. L'objectiu és oferir un pressupost clar, realista i professional que reflecteixi tant els costos d'implantació com els costos recurrents.

Aquest pressupost inclou:
- Costos inicials (pagament únic)
- Costos mensuals/anuals (manteniment i serveis)
- Justificació dels preus aplicats

## 🧾 3. Justificació de la proposta

### 💡 Preu per hora
El preu de **25 €/hora** s'ha establert segons tarifes habituals de tècnics informàtics junior/freelance a Espanya, que solen oscil·lar entre 20 € i 40 €/hora.

### ☁️ Elecció del SaaS
S'ha seleccionat **Google Workspace** perquè:
- És econòmic per PIMES
- Inclou correu, Drive, Meet i Docs
- Alta seguretat i disponibilitat
- Fàcil migració i ús

### 🌐 Hosting
S'ha optat per una solució econòmica perquè:
- La web és corporativa (no requereix alt rendiment)
- GitHub Pages pot reduir cost a 0 €

### 🛠️ Manteniment
El cost inclou:
- Prevenció d’errors
- Actualitzacions de seguretat
- Assistència tècnica bàsica

Aquest model assegura estabilitat i tranquil·litat al client.

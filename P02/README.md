# P02: Web corporativa

## Introducció

A la **tasca 8** vau triar quina seria la proposta de pàgina web que presentareu davant els responsables de **FoodLogístic S.A.** per substituir la seva pàgina actual.  
Toca, doncs, desplegar aquesta solució per tal que sigui visible pel client.

## Descripció de l’activitat

1. Creeu una carpeta anomenada `docs` a l’arrel del repositori del vostre projecte (en local).  
2. Copieu dins els arxius corresponents a la **versió definitiva** de la web.  
3. Un cop hàgiu confirmat els canvis i els hàgiu pujat a **GitHub**, configureu el **desplegament a GitHub Pages** del contingut de `docs`.  
4. Tots els membres de l’equip han de realitzar aquests passos als seus respectius repositoris.
